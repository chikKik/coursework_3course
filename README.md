# Blood Unit Converter

Веб-приложение для пересчета лабораторных единиц анализа крови и витаминов. Позволяет конвертировать показатели между различными системами измерения (мг/дл → ммоль/л, нг/мл → нмоль/л и др.).

## Возможности

- Пересчет 20+ лабораторных показателей и витаминов
- Поддержка популярных единиц измерения (мг/дл, ммоль/л, нг/мл, нмоль/л, мкг/дл, мкмоль/л и др.)
- Автоматический расчет по заданным коэффициентам
- Простой и интуитивно понятный веб-интерфейс
- Справочная таблица с коэффициентами пересчета
- Полная контейнеризация с Docker
- Автоматическое получение и обновление SSL-сертификатов Let's Encrypt

## Технологии

| Компонент | Технология |
|-----------|------------|
| Backend | Python 3.11 + Flask |
| WSGI-сервер | Gunicorn |
| Reverse-proxy | nginx-proxy (jwilder/nginx-proxy) |
| SSL-сертификаты | Let's Encrypt + acme-companion |
| Контейнеризация | Docker |
| Оркестрация | Docker Compose |

## Установка и запуск

### Требования

- Docker (версия 20.10 или выше)
- Docker Compose (версия 2.0 или выше)
- Сервер с публичным IP-адресом (для HTTPS)
- Зарегистрированный домен (для HTTPS)

### Локальный запуск (без HTTPS)

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/blood-converter.git
cd blood-converter

# Сборка и запуск
docker compose up -d

# Открыть в браузере
http://localhost:5000
